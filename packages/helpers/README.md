# @shinju-date/helpers

汎用的なユーティリティ関数を提供するパッケージ。

## 構成

- **TypeScript**: 型安全なユーティリティ関数
- **Next.js**: フレームワーク統合

## 使用方法

```typescript
import { helperFunction } from '@shinju-date/helpers'

// ヘルパー関数の使用例
const result = helperFunction(input)
```

## テスト

```bash
pnpm run test
```

## 開発

```bash
# ビルド
pnpm run build

# 開発モード（ウォッチ）
pnpm run dev
```
